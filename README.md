# ⚔️ GUANTY SRINP 
### ПРИКЛЮЧЕНЧЕСКАЯ ИГРА ПО МОТИВАМ MINECRAFT

Это хардкорная консольная RPG, написанная на **чистом C++** с использованием библиотеки `ncurses`. Никаких тяжелых фреймворков — только код, только логика!

---

## 🔥 Особенности
*   **Полностью на C++**: Максимальная производительность.
*   **Система состояний**: Меню, открытый мир, интерьеры домов и диалоги.
*   **NPC**: Взаимодействуй с жителями мира.
*   **Скины**: Выбирай свой уникальный стиль перед началом приключения.

---

## 🛠 Запуск проекта

Для запуска тебе понадобится компилятор `g++` и установленная библиотека `ncurses`.

### Сборка и старт:
```bash
g++ main.cpp -lncurses -o game
./game
