⚔️ GUANTY SRINP: Хардкорная RPG в стиле Minecraft на чистом C++

### Добро пожаловать в мир GUANTY SRINP – захватывающее приключение в жанре RPG, вдохновленное миром Minecraft!

Эта игра перенесет вас в хардкорный консольный мир, созданный с нуля на **чистом C++** и использующий мощь библиотеки `ncurses`. Здесь нет места излишним фреймворкам – только чистая логика и глубокое погружение в игровой процесс!

---

## 🔥 Ключевые особенности

### 🚀 **Полностью на C++**:
Максимальная производительность и полный контроль над кодом. Ощутите всю мощь и скорость, которую может дать чистый C++.

### 🗺️ **Динамичная система состояний**:
Переключайтесь между разнообразными игровыми локациями:
*   Бескрайний открытый мир
*   Уютные интерьеры домов
*   Оживленные диалоги с персонажами
*   И многое другое!

### 🧑‍🤝‍🧑 **Живой мир с NPC**:
Взаимодействуйте с обитателями этого мира, каждый из которых имеет свою роль и характер. Открывайте новые квесты и секреты.

### 🎨 **Персонализация персонажа**:
Создайте своего уникального героя, выбрав подходящий скин перед тем, как отправиться в путь. Выразите свой стиль!

---

## 🛠 Как начать играть

Для запуска **GUANTY SRINP** вам потребуется:
*   Компилятор `g++`
*   Установленная библиотека `ncurses`

### ⚙️ Процесс сборки и запуска:

1.  **Клонируйте репозиторий (если еще не сделали):**
    ```bash
    git clone <URL_вашего_репозитория>
    cd GUANTY_SRINP # Или название вашей папки
Скомпилируйте проект:

g++ main.cpp -lncurses -o game
-lncurses: Подключает библиотеку ncurses.
-o game: Создает исполняемый файл с именем game.
### Запустите игру:

./game
